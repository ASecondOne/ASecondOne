## What is Nyrana?

Nyrana is my long-term personal project to build my own ecosystem of tools and services.

Instead of relying on many different platforms, I want a unified system that runs on my own server and connects everything together.

Planned modules include things like:

- **Nyra** – my own Git-inspired source control
- **NyraMusic** – a self-hosted music library and player
- **Tasks & Calendar** – interconnected productivity tools
- **Automation and system integrations**
- more experimental modules in the future

Everything is designed to work together and run on a private home server.
